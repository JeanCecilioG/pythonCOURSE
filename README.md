# 100 Days of Code — Python

Repositório de estudo do curso **"Master Python by building 100 projects in 100 days"**.

Não estou aprendendo Python do zero. Estou usando o curso como estrutura para **reforçar fundamentos, revisar padrões e aprofundar áreas específicas da linguagem** que se conectam com outras stacks que já trabalho.

---

## Contexto

Python aparece em camadas diferentes do que me interessa: automação, dados, APIs, IA e integração entre sistemas. Este repositório acompanha o processo de consolidar isso de forma sistemática, projeto a projeto.

O foco não é completar o curso. É extrair o que importa de cada bloco temático.

---

## Estrutura

```
/aulas
  /aula-001
  /aula-002
  ...
/projects

  /dados
  /web
  /jogos
  /ia
```

Cada pasta de dia contém o projeto do dia e, quando relevante, anotações sobre o que foi revisado ou aprofundado.

## O que estou revisando e aprofundando

- **Automação** — uso de Python para tarefas que elimino de outras stacks
- **Dados** — Pandas e NumPy como base para análise e modelagem
- **APIs e Web** — Flask e integração com sistemas externos
- **IA aplicada** — conexão com o que já pesquiso em sistemas inteligentes
